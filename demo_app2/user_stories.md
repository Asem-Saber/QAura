# E-Learning Platform - User Stories

## Student Persona

- **As a Student**, I want to be able to sign up for an account so that I can access the e-learning platform.
- **As a Student**, I want to securely log in to my account so that I can manage my profile and courses.
- **As a Student**, I want to browse a catalog of available courses so that I can find topics I am interested in.
- **As a Student**, I want to view detailed information about a specific course so that I can decide whether to request enrollment.
- **As a Student**, I want to submit a request to enroll in a course so that I can gain access to its materials.
- **As a Student**, I want to provide a reason for my course request so that the admin knows why I want to join.
- **As a Student**, I want to view my profile information so that I can ensure my details are up to date.
- **As a Student**, I want to see a list of courses I am currently enrolled in so that I can track my learning progress.
- **As a Student**, I want to log out of the platform securely so that my session is terminated.

## System Admin Persona

- **As a System Admin**, I want to manage courses (create, update, delete) via the Django admin panel so that the course catalog remains current.
- **As a System Admin**, I want to review course enrollment requests from students so that I can approve or reject them based on availability or suitability.
- **As a System Admin**, I want to manage student accounts so that I can provide support and maintain platform security.
